Automated Team Formation

Instructions

Please select an event type to see specific instructions.

SQL Bootcamp
  1. Upload an Excel file with student data in the following format:
    • Timestamp
    • Email Address
    • Full Name
    • Track (SDET/DA)
    • Batch No
    • Course Type (Advanced/Full)
  2. The application will create teams based on course type and track:
    • Advanced course students are divided equally in 5-member teams
    • Full course students are formed in 5-member teams with balanced DA/SDET distribution
Selenium Hackathon
  1. Upload an Excel file with student data in the following format:
    • Timestamp
    • Email Address
    • Full Name
    • Track with Batch No
    • Are you working?
    • Time Zone
    • Have you completed DSAlgo Project
    • Have you participated in previous hackathon
  2. The application will create teams with these rules:
    • Teams will have a balanced size (approximately 5 students per team)
    • Students with previous hackathon experience are distributed evenly across teams
    • Working students are distributed evenly across teams
    • Teams are formed considering time zone compatibility (EST with CST, CST with PST)
